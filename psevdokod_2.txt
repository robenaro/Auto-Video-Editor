0. Описание документа
0.1 Есть документ, который состоит из таблицы. В ней столбцы: "таймкод", "название действия", 
    "текст". Эти столбцы есть в каждой строке таблицы.
    0.1.1 "Название действия" это какой-то текстовый объект. У него есть соответствующие ему 
    значения "таймкод" и "текст", находящиеся в соседних столбцах на той же строке. 
            В этом столбце есть неизменяемые постоянные значение: "удалить с", "удалить по", 
            "тизер с", "тизер по", "вставить", "нешаблонный".
        0.1.1.2 У значения "вставить" есть связанный с ним дополнительный столбец "тип вставки".
        В нем есть неизменяемые постоянные значения: "слайд", "слайд боковой", "слайд полный", 
        "плашка пустая", "плашка миф", "плашка книга" .
            0.1.1.1.2 У значений "слайд боковой" и "слайд полный" есть дополнительное значение, 
            хранящееся в столбце "доп параметр". 
            0.1.1.1.3 У значения "плашка книга" есть дополнительное значение, хранящееся в столбце 
            "доп параметр". 
            0.1.1.1.4 В зависимости от созданных слайдов могут понадобиться другие дополнительные 
            значения в дополнительных столбцах. (уже в новых версиях)
0.2. "Таймкод" - это текстовый объект в виде "01:17:25:00", где "часы:минуты:секунды:кадры". 

1.1 "Текст" - это какой-то текстовый объект. 

1. Таблица будет написана в Google.таблице. Можно экспортировать .csv и там будет 
последовательность значений через запятую.

Доступ к конкретному значению можно получить с помощью ссылок на индексы, где индекс равен "a"*"b". 
"а" = столбец:№1, №2, №3 и т.д., а "b" = номер строки. 
Нужно из последовательности создать список и рассортировать его по группам.
    1.1  Рассмотрим столбец "название действия". В нем есть постоянные значения "удалить с" и 
        "удалить по".
        У значения "удалить с" есть привязанный к нему первый таймкод (timecod1), находящийся на 
        той же строке в столбце "таймкод", и привязанный к нему текст, находящийся на той же строке в столбце "текст".
        У значения "удалить по" есть второй таймкод (timecode2) и текст, находищеся аналогично 
        перовому.
        1.1.1 Код читает ячейки столбца "название действия":
            если ему встречается функция "удалить с", то он сохраняет соответствующий ему таймкод 
            в массиве под индексом [0] и соответствующие ему текст в массиве под индексом [2].
            если ему встречается функция "удалить по", то он сохраняет соответствующий ему таймкод 
            в массиве под индексом [1] и соответствующие ему текст в массиве под индексом [3].
            В итоге должен получиться список с парами таймкодов 
            var удалить = [[timecod1,timecode2,text1, text2], [timecode1, timecode2, text1, text2], ... ] 
        1.1.2 (в данный момент подходящая форма для таймкода это - последовательность из 8 цифр, 
        полученная в результате удлаение всех знаков кроме цифр)

1.2 Аналогично пункту 1.1 Рассмотрим столбец "название действия". В нем есть постоянные значения 
"тизер с" и "тизер по".
    У значения "тизер с" есть привязанный к нему первый таймкод (timecod1), находящийся на той же
     строке в столбце "таймкод", и привязанный к нему текст, находящийся на той же строке в столбце
      "текст".
    У значения "тизер по" есть второй таймкод (timecode2) и текст, находищеся аналогично перовому.
        1.2.1 Код читает ячейки столбца "название действия":
            если ему встречается функция "тизер с", то он сохраняет соответствующий ему 
            таймкод в массиве под индексом [0] и соответствующие ему текст в массиве под индексом [2].
            если ему встречается функция "тизер по", то он сохраняет соответствующий ему 
            таймкод в массиве под индексом [1] и соответствующие ему текст в массиве под индексом [3].
            В итоге должен получиться список с парами таймкодов 
            var тизер = [[timecod1,timecode2,text1, text2], [timecode1, timecode2, text1, text2], ... ] 
        1.2.2 (в данный момент подходящая форма для таймкода это - последовательность 
        из 8 цифр, полученная в результате удлаение всех знаков кроме цифр)

1.3 Код читает ячейки столбца "название действия" и если ему встречается функция 
    "вставить", то он сохраняет
    1.3.1 соответствующий ему таймкод в массиве под индексом [0]
    1.3.2 соответствующие ему текст в массиве под индексом [1]
    1.3.3 соответствующий ему значение из столбца "тип вставки" под индексом [2]
    1.3.4 соответствующие ему значения из дополнительных столбцов если они есть. 
    (когда будут понятны все слайды, то они будут прописаны)
        В итоге должен получиться список var вставить = [[timecode1, "текст", 
        "название вставки","заголовок", "доп параметр"], [...], ... ]

1.4 код проходит по функциям и если ему встречается функция "нешаблонный", то он 
сохраняет в список "нешаблонный" значение True и значение соответствующего ему таймкода. 
    В итоге должен появиться список "нешаблонный" = [[timecode, True], [timecode, True], ...]

1. Удаление
    1.1 Убираются знаки из всех значений timecod1 и timecod2 списка удалить.
        удалить = [[timecod1,timecode2,text1, text2], [timecode1, timecode2, text1, text2], ... ] 
    1.2 выполнить цикл пока не закончатся значения в списке удалить:
        1.2.1 Взять значение timecod1 из каждого внутреннего списка удалить (т.е. 
        удалить[0][0], удалить[1][0], удалить[2][0] и т.д.) и подставить его под 
        встроенную функцию "сделать разрез".
        1.2.2 Аналогично для таймкода 2. Взять значение timecod2 из каждого внутреннего 
        списка удалить (т.е. удалить[1][1], удалить[1][1], удалить[2][1] и т.д.) 
        и подставить его под встроенную функцию "сделать разрез".
        1.2.3 Удалить ВИДЕОклип между этими 2-умя разрезами с помощью функции .remove
        1.2.4 Удалить АУДИОклип между этими 2-умя разрезами с помощью функции .remove

2. Вырезать клип
    2.1 Убираются знаки из всех значений timecod1 и timecod2 списка удалить.
        удалить = [[timecod1,timecode2,text1, text2], [timecode1, timecode2, text1, text2], ... ] 
    2.2 выполнить цикл пока не закончатся значения в списке удалить:
        2.2.1 Взять значение timecod1 из каждого внутреннего списка удалить 
        (т.е. удалить[0][0], удалить[1][0], удалить[2][0] и т.д.) и подставить его 
        под встроенную функцию "сделать разрез".
        2.2.2. Удалить ВИДЕОклип перед этим разрезом с помощью функции .remove
        2.2.3. Удалить АУДИОклипперед этим разрезом с помощью функции .remove
        2.2.4 Аналогично пункту 2.2.1. Взять значение timecod2 из каждого 
        внутреннего списка удалить (т.е. удалить[1][1], удалить[1][1], удалить[2][1] 
        и т.д.) и подставить его под встроенную функцию "сделать разрез".
    2.3 Удалить ВИДЕОклип перед этим разрезом с помощью функции .remove
    2.4 Удалить АУДИОклипперед этим разрезом с помощью функции .remove

3. Нешаблонный
3.1 Код считает сколько значений в списке нешаблонный и сообщает пользователю 
количество этих значений. 
    (и (если получится) ставить красные маркеры на место таймкодов из списка)


4. Вставить. Есть список var вставить = [[timecode1, "текст", 
    "название вставки","доп параметр"], [...], ... ]
    4.1 Переводим значение "timecode1" из формата 00:02:04:00, 
    (где часы:минуты:секунды:кадры) в формат секунд. 
    4.2 Если "название вставки" = "слайд", то посчитать количесто символов: 
        4.2.4. если оно больше зданного числа, то
            4.2.1.1"название вставки" = "слайд полный",
            4.2.1.2 иначе "название вставки" = "слайд боковой"
    4.3 Выполнить цикл, пока количество значений списка "вставить" не закончится.
            4.3.1 Вставляем слайд на таймлайн используя функцию 
                sequence.importMGT, где нужен timecode1, "название вставки" 
                и путь к нему (это константа). 
                mogrtImport(File("G:/Монтаж/Ливрезон/ЛИМ/MOGRT LIM/" + (название вставки) + ".mogrt")); 
                Путь = константа + "название слайда"
            4.3.2  Если название вставки "слайд боковой" или "слайд полный", 
                то с помощью функции .getMGTComponent().properties[]
                4.3.2.1 Если "доп параметр" = "задача №N", то
                    4.3.2.1.1 В переменной вести подсчет количества слайдов 
                    этого типа, начиная с единицы. Параметр 
                    с индексом [n] = "Задача №" + (значение этой переменной) +":".
                    4.3.4.1.2 Иначе параметра с индексом [0] этого клипа = "доп параметр"
                4.3.2.2 заполнить значение параметра с индексом [1] этого клипа значением "текст".
            4.3.3 Если "название вставки" = "плашка пустая", то 
                4.3.2.1. выполнить значение параметра с индексом [0] этого клипа = "пустой"
                4.3.2.2 заполнить значение параметра с индексом [1] этого клипа значением "текст".
            4.3.4 Если "название вставки" = "плашка миф", то
                4.3.4.1 выполнить значение параметра с индексом [0] этого клипа = "миф"
                4.3.4.2 заполнить значение параметра с индексом [1] этого клипа значением "текст".
                4.3.4.3 В переменной вести подсчет количества 
                    слайдов этого типа, начиная с единицы. Вводить в параметр 
                    с индексом [n] значение этой переменной.
            4.3.5 Если "название вставки" = "плашка книга", 
                4.3.5.1 выполнить значение параметра с индексом [0] этого клипа = "книга".
                4.3.5.2 Выполнить: значение параметра с индексом [n] 
                        этого клипа = "дополнительный параметр".
                4.3.5.3 Выполнить: значение параметра с индексом [n]  этого клипа= "текст"
